# Agent Skills Directory

**Centralized for multi-agent use (Claude primary)**

This directory contains skills organized for use across multiple AI agents, with Claude as the primary agent.

## Structure

### Standard Skills (Pre-installed)
- **algorithmic-art** - Creating algorithmic art using p5.js
- **brand-guidelines** - Anthropic brand colors and typography
- **canvas-design** - Canvas design tools and utilities
- **doc-coauthoring** - Structured documentation workflow
- **docx** - Document creation and editing
- **mcp-builder** - MCP server development guide
- **pdf** - PDF manipulation toolkit
- **pptx** - Presentation creation and editing
- **skill-creator** - Guide for creating effective skills
- **slack-gif-creator** - Animated GIFs optimized for Slack
- **theme-factory** - Styling artifacts with themes
- **web-artifacts-builder** - Complex HTML artifacts with React/Tailwind
- **webapp-testing** - Local web application testing with Playwright
- **xlsx** - Spreadsheet creation and analysis

### Project-Specific Skills
- **core/** - NinjaTrader strategy development fundamentals
- **project-specific/** - Universal OR Strategy context and implementation details
- **references/** - Live price tracking and market data references
- **trading/** - APEX/Rithmic trading system, session timezones, code review guidelines
