# Gemini Skills & Context

This directory stores persistent context and skills for Gemini to assist with the UniversalORStrategy project. It mirrors the structure used by other AI assistants to ensure continuity.

## Structure
- `skills/`: Reusable knowledge bases and patterns.
  - `core/`: General NinjaTrader 8 and C# patterns.
  - `trading/`: Specific trading strategies (ORB, RMA, etc.).
  - `project-specific/`: Context for this specific codebase and session.
  - `references/`: API docs, critical bug fixes, and specifications.
  - `changelog/`: History of major changes and learnings.
- `context/`: Dynamic session information and current focus.

## Usage
Gemini uses these files to maintain context across sessions and ensure adherence to project standards, specifically Apex compliance and memory efficiency.
